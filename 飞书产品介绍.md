# 飞书（Feishu）产品介绍

## 📖 什么是飞书？

飞书是字节跳动推出的一站式企业协作与管理平台，整合了即时通讯、日历、文档、云盘、视频会议等多种办公工具，旨在提升团队协作效率。

---

## 🎯 核心功能

### 1. 即时通讯（IM）
- **高效沟通**：支持单聊、群聊、话题讨论
- **消息回复**：支持引用回复、表情回应、消息点赞
- **富媒体**：支持图片、文件、链接预览
- **已读未读**：实时查看消息阅读状态
- **机器人**：支持自定义机器人接入

### 2. 飞书文档（Docs）
- **在线协作**：多人实时编辑，自动保存
- **丰富格式**：支持表格、代码块、流程图、看板等
- **模板库**：提供会议纪要、项目计划等模板
- **评论批注**：支持@同事、添加评论
- **版本历史**：随时查看和恢复历史版本

### 3. 飞书日历（Calendar）
- **日程管理**：个人日程、团队日程一目了然
- **会议预约**：一键创建会议，自动发送邀请
- **忙闲查看**：查看同事空闲时间，方便约会
- **会议提醒**：支持多种提醒方式

### 4. 飞书云盘（Drive）
- **文件存储**：云端存储，随时随地访问
- **共享协作**：文件夹共享、权限管理
- **在线预览**：支持多种格式文件在线预览
- **版本管理**：文件历史版本可追溯

### 5. 飞书会议（Meetings）
- **视频会议**：高清视频，支持多人参会
- **屏幕共享**：演示文档、代码、桌面
- **会议录制**：自动录制，生成文字纪要
- **虚拟背景**：支持自定义背景

### 6. 飞书审批（Approval）
- **流程自动化**：请假、报销、采购等审批流程
- **自定义表单**：灵活配置审批字段
- **移动审批**：随时随地处理审批

### 7. 飞书工作台（Workplace）
- **应用集成**：集成第三方应用和企业自建应用
- **待办事项**：统一管理待处理任务
- **数据看板**：可视化展示业务数据

## 🔧 开发者能力

### 开放平台
- **应用开发**：创建企业内部应用或第三方应用
- **机器人**：通过 Webhook 或 WebSocket 接收事件
- **API 能力**：消息、文档、云盘、日历等完整 API
- **小程序**：轻量级应用，快速部署

### 常见 API 权限
- `im:message` - 消息收发
- `docx:document` - 文档操作
- `drive:drive` - 云盘文件管理
- `wiki:wiki` - 知识库管理
- `contact:user` - 联系人信息
